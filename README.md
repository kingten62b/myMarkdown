# 1.标题 [数个 "#" + 空格]

```

## 二级标题
### 三级标题
```

**若在Typora开启严格模式,则必须在"#"后加空格**

# 6.代码块 [用 "```" 包围]

```

    print(“Hello, world!\n");
    retrun O;
}
```



````c
#include <stdio.h>

    print("Hello, world!\n");

````
